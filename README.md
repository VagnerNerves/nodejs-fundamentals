## 🚀 Running the Project

### Back-end

Clone the project

```bash
  git clone https://github.com/VagnerNerves/nodejs-fundamentals-ignitenode.git
```

Enter the project directory

```bash
  cd nodejs-fundamentals-ignitenode
```

Start the server

```bash
  npm run dev
```
